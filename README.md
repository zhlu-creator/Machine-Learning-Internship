# Machine-Learning-Internship

## 高光谱遥感地物分类与降维可视化

技术栈： Python、pandas、scikit-learn（LogisticRegression、SVM、LDA、PCA）、Matplotlib

具体工作：预处理5个含中文文件名的CSV数据集，调整pandas读取参数并清洗首行列名；实现基于Band1和Band31的散点图可视化，分析5类地物（grass、path、road、roof、tree）的可分性；采用一对多策略，使用对率回归、支持向量机、线性判别分析进行监督多分类（5类）与二分类（透水面/不透水面），分别训练5个/10个/5个分类器及1个二分类器，并生成混淆矩阵、召回率、查准率；使用PCA将191维数据降维至90维，并再次可视化降维后Band1与Band31的类别分布。

量化成果：对率回归多分类：平均召回率0.970，平均查准率0.968；二分类AUC=0.9999；支持向量机多分类：平均召回率0.955，平均查准率0.944；二分类AUC=0.9999；线性判别分析多分类：平均召回率0.876，平均查准率0.839；二分类AUC=0.9994；PCA降维至90维，成功压缩特征空间并评估降维后类别可分性变化。
